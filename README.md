# Prompt-a-thon-2026 Base Game

Welcome to **Prompt-a-thon 2026!** This is the ultimate starting point for your hackathon project. You are provided with a simple, functional Mario-style platformer game base. 

Your goal is to use your imagination and AI coding agents to build upon this foundation. Transform it into something amazing, expand on its mechanics, overhaul the design — the choice is yours.
However, your challenge is to preserve the essence of the game by ensuring it remains a 2D experience with continuous side-scrolling movement, similar to classics like Super Mario or the Chrome Dino game. You are free to change the character and theme to whatever you want. Show us a game we've never seen before!!

Further , it is IEEE Computer Society's 80th Anniversary.
Bonus Points for adding themes or elements related this remarkable milestone. 

## Judging Criteria

Your final submission will be evaluated based on the following metrics:

- **Complexity & Functionality (30%)**: Does the game work? Is the logic sound and free of game-breaking bugs?
- **AI Tool (Antigravity) Mastery (20%)**: Effective use of the Agent Manager, distinct LLM selection, MCP (Model Context Protocol) integrations, and other AI tooling capabilities.
- **Creativity & Innovation (30%)**: Originality of the idea. How much have you transformed or elevated the base template? (inclusion of CompSoc's 80th anniversary elements) 
- **UI/UX (20%)**: Visual appeal, polish, and overall user experience.

Good luck, have fun, and let your creativity run wild!

# or
